# 🍪 Cookie Clicker

A simple cookie clicker game built with HTML, CSS, and vanilla JavaScript.

## 🚀 Features
* **Clicking:** Click the giant cookie to earn cookies.
* **Store:** Buy buildings to automate your cookie production.
* **Scaling Prices:** Building prices increase exponentially with each purchase.
* **Visuals:** Shows your owned buildings visually in the middle section and features a rolling news ticker.

## 🛠️ Tech Stack
* HTML5
* CSS3
* JavaScript (Vanilla)

## 💻 How to Play
1. Download or clone this repository.
2. Open `index.html` in any web browser to play!
